# BioAttend – iOS Biometric Attendance System

## Overview
BioAttend is a conceptual iOS-based attendance system that enables employees to securely mark attendance using biometric authentication such as fingerprint or facial recognition.

The application eliminates the need for physical biometric devices, promoting a contactless and efficient attendance process.

## Features
- Biometric authentication (Face ID / Fingerprint)
- Secure attendance marking
- Contactless system design
- Simple and intuitive user interface

## Technologies Used
- Swift
- iOS development concepts
- Biometric authentication (conceptual implementation)
